Academic Insight

📌 Description
This is a student academic analysis system built using Python and Flask.

🚀 Features
- Student performance prediction
- User Authentication System (Login/Logout)
- Separate Admin Panel with advanced controls
- Student performance prediction using ML model
- Data management for student records
- Web-based interface using Flask
- Data analysis using ML model
- Web interface using Flask

🛠️ Technologies Used
- Python
- Flask
- Machine Learning
- SQLite

▶️ How to Run
1. Clone the repo
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python app.py

📂 Project Structure
- app.py → main app
- model.pkl → trained ML model
- templates/ → HTML files
- static/ → CSS/JS
